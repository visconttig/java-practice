# Exercise text

### English
Read a value $n$. Calculate and write its respective $\textit{factorial}$.

> Factorial of $n$:
>> $$ n! = n * (n - 1) * (n - 2) * (n - 3) * \dots * 1 $$

Remember that, by definition, ***the factorial of 0 is 1***.
